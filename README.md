# APK Forensic Toolkit (AFT)

أداة متقدمة للهندسة العكسية وتحليل أمن تطبيقات Android.

## المميزات

- ✅ تفكيك APK باستخدام APKTool وJadx
- ✅ تحليل أمني شامل للثغرات
- ✅ إعادة بناء وتوقيع التطبيقات
- ✅ إنشاء تقارير احترافية (MD/HTML/PDF)
- ✅ واجهة سطر أوامر سهلة الاستخدام
- ✅ دعم أنظمة Linux/Windows/macOS
